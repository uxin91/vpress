# http 执行过程

# 请求响应流程（浏览器做了什么）

  1. 先在本地host文件里找对应的域名对应的ip
  2. 在去dns服务器里找com的域，并返回ip
  3. 通过在dns拿到的ip，在本地临时开一个端口，和服务器的端口（8080)建立起一个连接
  4. 浏览器会将本地信息打一个包，通过那个随机端口和服务器做连接
  5. 拿到相应报文后，找到content type类型
  • 如果是html则直接渲染
  • 如果是css，js则再请求这个流程

# 请求响应流程（服务端）

  1. 打开客户端提交来的包（请求报文）
  2. 根据请求路径，找到对应的mime type
  3. 判断是否为静态文件。如果是html则自己解决。如果是动态文件php,jsp 则交由对应的软件，比方说php去做
  4. 将服务端内容打成一个包（相应报文）

# php连接mysql

  • 创建连接： $conn=mysqli('localhost','root','123456','demo'); ip-用户名-密码-数据库名
  • 设置编码：必须要在查询之前设置 mysqli_set_charset($conn,'utf8')
  • 查询 $query=mysqli_query($conn,'select * from table;');
  • 数组遍历 while($item=mysqli_fetch_assoc($query))

如果是删除和添加操作，还需要加操作所影响的记录行数

  • $affected_rows=mysqli_affected_rows($conn);
<?php
//简历连接
