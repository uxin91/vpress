# mock的基本使用

  • mock官方文档 (opens new window) 生成随机数据，拦截 Ajax 请求

  • import Mock from 'mockjs'引入

生成一个最简单的结构,得到一个随机id

let a = Mock.mock('@id')
console.log(a);   //710000201608218111

随机创建类似json的对象模版

let arr=[]
    for(let i=0;i<10;i++){
        arr[i]=
         Mock.mock(
             {
            id:'@id()',
            username:'@cname()',
            date:'@date()',
            avatar:'@image("200x200","red","#fff","avatar")', //参数，图片尺寸，背景色，文字颜色，内容
             tip:'@dataImage()', //图片随机生成
            dec:'@paragraph()',
            ip:'@ip()',
            email:'@email()'
        }
        )
    }
   console.log(arr);
