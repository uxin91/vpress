# 父子组件传值

还是和vue一样，组件通信也是必须要掌握的。

  • 父->子 在父组件中引入子组件，然后在标签上定义
  • 子组件接收方式：通过props <span>{this.props.msg}</span>
  • 传方法也类似 <Header fat={this.run}></Header>
  • 子组件同样根据props获取 <button onClick={this.props.fat}>获取</button>

  • 父组件还可以把自己全部传过去，<Header homes={this}></Header>

  • 子组件通过<b>{this.props.homes.state.name}</b>接收

那么子组件传父组件呢，子组件里也无法引入父组件作为标签？ 那么就要注意，子组件传父组件还是要基于父组件将自己全部传过来的情况下，通过调父组件的方法，bind子组件的信息进行传递

父组件设置（基于之前父组件将自己全部传给子组件）并选择用此方法接收子组件传过来的title
 getChilds=(res)=>{
        console.log(res.state.title);
    }

子组件设置（可以在子组件中定义一个方法，进行信息的绑定）：

子组件设置（定义一个传值方法）
<button onClick={this.sendmess}>子组件的title穿给父组件</button>
方法：
sendmess=()=>{
        this.props.homes.getChilds(this,this.state.title)
    }

# react路由

  1. 引入import {BrowserRouter as Router,Route,Link} from 'react-router-dom';
  2. 引入路由的子模块 import homes from .....
  3. router标签包裹以下两项
<Link to ='/'>....</Link>
<Route exact path='/' component={homes} />

WARNING

exact为严格模式匹配。如果不加的话，那么初始时进入的值清不掉

# 嵌套路由

  • 引入路由：import {BrowserRouter as Router,Route,Link} from 'react-router-dom';

  • 引入子组件

  • 填写路由规则,注意用Router包裹

              <ul>
              <li><Link to='/shouye'>首页</Link></li>
              <li><Link to='/detal/usr'>详情</Link></li>
          </ul>

         
          <main>
              <div className="left">
                  <Route path='/detal/usr' component={Deal}/>
              </div>
              <div className="right">
                  <Route path='/shouye' component={Shouye}/>
              </div>
          </main>

这里的感觉就是，vue里面占位的是router-view。而react里面直接用route

<Route path='/...' component={...}/>

# Ant-ui基本使用

  • 安装 $ npm install antd --save
  • 引入样式 修改 src/App.css，（或者Index.css）在文件顶部引入 antd/dist/antd.css
@import '~antd/dist/antd.css';

.App {
  text-align: center;
}
  • 看文档，按需使用

以表单为例

  1. 在需要引入的页面引入需要的组件的依赖 import { Form, Input, Button, Checkbox } from 'antd';
  2. 复制代码
  3. render中调用
    render() {
        return (
              <div>
                  <Demo />
              </div>
        )
        }

demo这个标签和上面定义的名字要相同

const Demo = () => {...}

完整代码如下：

import React,{Component} from 'react';
import { Form, Input, Button, Checkbox } from 'antd';
const layout = {
    labelCol: {
        span: 8,
    },
    wrapperCol: {
        span: 16,
    },
};
const tailLayout = {
    wrapperCol: {
        offset: 8,
        span: 16,
    },
};

const Demo = () => {
    const onFinish = values => {
        console.log('Success:', values);
    };

    const onFinishFailed = errorInfo => {
        console.log('Failed:', errorInfo);
    };

    return (
        <Form
            {...layout}
            name="basic"
            initialValues={{
                remember: true,
            }}
            onFinish={onFinish}
            onFinishFailed={onFinishFailed}
        >
            <Form.Item
                label="Username"
                name="username"
                rules={[
                    {
                        required: true,
                        message: 'Please input your username!',
                    },
                ]}
            >
                <Input />
            </Form.Item>

            <Form.Item
                label="Password"
                name="password"
                rules={[
                    {
                        required: true,
                        message: 'Please input your password!',
                    },
                ]}
            >
                <Input.Password />
            </Form.Item>

            <Form.Item {...tailLayout} name="remember" valuePropName="checked">
                <Checkbox>Remember me</Checkbox>
            </Form.Item>

            <Form.Item {...tailLayout}>
                <Button type="primary" htmlType="submit">
                    Submit
                </Button>
            </Form.Item>
        </Form>
    );
};

class Shouye extends Component{

    constructor(props) {
        super(props);

    }
    render() {
        return (
              <div>
                  <Demo />
              </div>
        )
        }

};

export default Shouye;
