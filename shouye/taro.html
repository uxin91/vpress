基本上通过异步方式去获取

轮播图这块，将数据传至子组件上。轮播图的组件小程序内部自己的方法，按要求引入对应的组件，设置好属性就可以直接用了

父组件
<Swipers sendsp={this.state.swipers} />

子组件

import {View,Swiper,SwiperItem,Navigator,Image} from "@tarojs/components";

  render() {
    let ele = this.props.sendsp;
    let elements = ele.map((v,k)=>{
      return (
        <SwiperItem key={k}>
          <Navigator>
            <Image src={v.img}></Image>
          </Navigator>
        </SwiperItem>
      )
    })
    return(
      <View>
        <Swiper className='sps'
                autoplay={true}
                indicator-dots={true}
        >
        {elements}
        </Swiper>
      </View>
    )
  }

# 接收父组件的传值

父组件：<Cat say='父组件传值' />

子组件：cat1:this.props.say

修改值的方式，和react一样

this.setSate({
	cat1:‘修改值’
})

setState的方式是异步的（不会马上生效） 通过设置回调函数可以让其不是异步的

this.setstate({
...
},()=>{
console.log(...)
})

# taro中绑定事件的方式

onClick={this.fun}
fun(){.....}

在taro中，函数这块的写法不用于react要改变this指向(react中fun=()=>{...})

阻止事件冒泡 e.stopPropogation()

事件触发的同时传递参数,通过bind this.fun.bind(this,'传递的参数')
